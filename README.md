# 📦 Amazon Sales & Customer Reviews Analysis

## 📊 Dashboard Overview
![Amazon Dashboard](amazon_dashboard.png)


## 📌 Project Description
This project focuses on analyzing Amazon's sales data and customer reviews. I used **SQL** for database structuring, **Python** for data processing, and **Power BI** for the final visualization.

## 🛠 Tools Used
- **SQL:** Data querying and schema design.
- **Python:** Data cleaning and analysis.
- **Power BI:** Interactive dashboarding.

## 📁 Files in this Repo
- `POWER bi amazon.pbix`: Interactive Dashboard file.
- `SQL amazon_sales`: SQL scripts and queries.
- `python files`: Data processing scripts.
